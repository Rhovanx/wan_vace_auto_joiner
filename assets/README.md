# Assets Folder

Place your screenshots and images here:

- `workflow_preview.png` - Screenshot of the complete workflow
- `node_connections.png` - Diagram showing node connections
- `example_output.gif` - Example of joined video output (optional)

## Recommended Screenshot Sizes

- Workflow preview: 1200x800 px or larger
- Node close-ups: 600x400 px

## Creating Good Screenshots

1. Zoom to show the key nodes clearly
2. Use a clean ComfyUI theme
3. Highlight the connection paths if possible
